Welcome to CSS styling LexaN

LexaN is an open source toolkit witch gonna help you to build your web app or web site with HTML and CSS. Build your app to look nice very easy without Bootstrap or jQuery. LexaN was developing by @Li54nder since 2019.

Linking LexaN with your web page


This code type in head tag of HTML document: 
<link rel="stylesheet" href="LexaN.css">


And this in the end of body tag: 
<script src="LexaN.js"></script>
